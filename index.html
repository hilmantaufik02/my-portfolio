About Me

Hai nama saya Hilman Taufik, biasa dipanggil Hilman. Saya seorang mahasiswa berusia 25 tahun yang bercita-cita menjadi programmer handal. Saya menyukai coding sejak SMK karena tugas membuat sebuah aplikasi berbasis web dan saya mempunyai projek tugas akhir aplikasi klinik. Saya sedang mengejar gelar Sarjana Komputer di Universitas di Cimahi untuk memperdalam keterampilan saya dalam bidang pengembang web.

Tujuan utama dalam coding adalah untuk merancang dan membuat aplikasi web yang berguna di masa depan. Dalam hal itu saya seringkali mengasah kemampuan saya dan mencoba hal baru dalam membuat dan mengembangkan aplikasi web. Selain coding, saya biasa meluangkan waktu dengan bermain game di Hp, saya senang sekali bermain genre survival, petualangan dan juga simulasi.
